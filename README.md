# 가상화폐거래소 토이프로젝트 레파지토리

## 페이지

- 가상화폐 리스트
- 북마크 리스트
- 가상화폐 상세정보

## 패키지 버전

- react: 17.0.1
- react-router-dom: 5.2.0

## 구조

- components : 각 페이지별, 공통으로 사용되는 컴포넌트들이 위치
- config : app 기본 config 정의
- pages : 각 페이지의 페이지 컴포넌트 위치
- utils : 프로젝트에서 사용되는 utils들이 위치

## 프로젝트 실행

1. 패키지 인스톨

```
npm install
```

2. 로컬 서버 실행

```
npm run start
```

3. localhost:3000 접속
